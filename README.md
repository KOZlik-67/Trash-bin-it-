Trash 
Automatic trash bin that opens its lid when it detects motion.
----------------------------------------------------------------------------------
Components List-
Microcontroller: Arduino Uno or Nano
Sensor: HC-SR04 Ultrasonic Sensor
Actuator: SG90 Servo Motor
Power: 9V Battery or Power Bank
Casing: Custom 3D printed or plastic bin
----------------------------------------------------------------------------------
How to use
1 Connect components according to the wiring diagram.
2 Install the (Servo.h) library in Arduino IDE.
3 Upload (Trash.ino) to your board.

мусорка
Автоматическое мусорное ведро, которое открывает крышку при обнаружении движения.
----------------------------------------------------------------------------------
Список компонентов:
Микроконтроллер: Arduino Uno или Nano
Датчик: Ультразвуковой датчик HC-SR04
Привод (Актуатор): Сервопривод SG90
Питание: Батарейка 9В (Крона) или Power Bank
Корпус: Напечатанный на 3D-принтере или обычное пластиковое ведро
----------------------------------------------------------------------------------
Как использовать:
Подключите компоненты согласно схеме соединений.
Установите библиотеку (Servo.h) в среде Arduino IDE.
Загрузите файл (Trash.ino) на вашу плату.
